@dbid 523 @title Huevos 1.1b2 @link http://ranchero.com/software/huevos/beta.php @pubDate Thu Oct 24 12:00:45 -0700 2002 @modDate Thu Oct 24 12:01:46 -0700 2002 This new beta includes a very useful change—you can type a hotkey combination in any app and bring Huevos to the front instantly. You can change the combination via the Preferences window.

(Note to Cocoa developers: the HotKeys API is a Carbon API, and I didn’t know how to access it from Cocoa—until I saw this helpful post from the Unsanity folks.)

Huevos remains open source (BSD license). You can download the latest source code from the beta page.
