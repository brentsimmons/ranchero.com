@dbid 1334 @title Delivering Content with RSS for Web Developers on Mac OS X @link http://developer.apple.com/internet/deliveringcontentwithrss.html @pubDate Wed Apr 05 13:09:47 -0700 2006 @modDate Wed Apr 05 13:09:47 -0700 2006 Apple Developer Connection: “RSS, or Really Simple Syndication, is revolutionizing the delivery of web content. RSS is an XML (eXtensible Markup Language) format for coordinating the delivery of time-based content streams, or ‘feeds.’ This means that RSS can be used to deliver content that changes over time. RSS provides for the inclusion of additional data, similar to email attachments, using the <enclosure> tag.”
