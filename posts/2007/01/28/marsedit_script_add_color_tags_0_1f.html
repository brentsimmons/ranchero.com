@dbid 1530 @title MarsEdit Script: Add Color Tags 0.1f @link http://scriptbuilders.net/files/addcolortags0.1f.html @pubDate Sun Jan 28 17:04:08 -0800 2007 @modDate Sun Jan 28 17:04:08 -0800 2007 @categoryArray MarsEdit, MarsEdit-Scripts Flareman: “I wrote it to be able to add color to text in blog posts to Blogger, using the <span style="color:(r,g,b);"></span> tag that Blogger’s own HTML WYSIWYG editor uses.
